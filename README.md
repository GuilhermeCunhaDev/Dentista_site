 ## Some fields in main Model is:

+ Navbar
+ Hero
+ Service
+ About
+ Doctor
+ Call to Action
+ Blog
+ Footer

## Website objectives
+  Lead capture
+  Information about services
+  Information about staff
+  Provision of means of contact and location
+  Bring Tips and News
